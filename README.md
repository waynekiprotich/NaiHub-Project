# NaiHub-Project
NaiHub is a React-based web application designed to bridge the social gap in Nairobi. It allows users to discover local entertainment, add their own gatherings, and connect with other attendees via dedicated WhatsApp groups or links to buy tickets and more .

# Features
View Events Nearby Dynamic display of  sports, concerts, and more in Nairobi.
Browse events by category or specific interests by filtering .
Real-time submission form for users to list new gatherings.
Eternal planning which can be implimented from the link provided.

# Structure
`App.jsx` Main container managing central state. 
`Navbar.jsx` Global navigation (Home, Add Event, About). 
`EventList.jsx` Wrapper to map and display event data. 
`EventCard.jsx` Displays event details & WhatsApp redirect logic. 
`AddEvent.jsx` Handles form validation and new event submission. 
`FilterBar.jsx` UI controls for filtering by category or date.
